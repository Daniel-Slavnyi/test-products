# Тестове завдання Test Products :muscle: :wave:

Test Products - це веб-додаток з використанням React та Redux, який демонструє
створення списку товарів з можливістю фільтрації та сортування.

Опис Test Products дозволяє користувачам переглядати список товарів,
використовуючи категорії, фільтри та сортування. Кожен товар містить інформацію
про заголовок, опис, ціну, рейтинг та наявність на складі. Крім того,
користувачі можуть переглядати деталі кожного товару на окремій сторінці.

## :star: ЗАВДАННЯ :

1. Створіть додаток, що складається з декількох компонентів: заголовок, пошукове
   поле, список товарів, та форма для додавання нового товару.
2. Використовуйте React Router для створення маршрутів додатку, які дозволяють
   користувачам переходити між сторінками.
3. Використовуйте Redux для управління станом додатку. Створіть reducer та
   actions, які дозволяють додавати, видаляти та оновлювати товари.
4. Створіть компонент списку товару, який отримує дані з Redux store та
   відображає їх у вигляді таблиці з колонками: ID, назва, опис, ціна, фото,
   рейтинг, сток, категорія.
5. Додайте можливість сортування та фільтрації товару за кожною колонкою.
6. Додайте пошукове поле, яке дозволяє користувачам шукати товари за назвою або
   категорією. Компонент списку товару повинен оновлюватись автоматично, коли
   користувач вводить запит у пошукове поле.
7. Створіть форму для додавання нового товару. Використайте Formik та Yup для
   валідації введених даних. Форма повинна містити поля: Назва, Автор, Рік
   видання, Рейтинг.
8. Додайте можливість видалити елемент зі списку та з Redux store.

## :exclamation: Як запустити додаток:

1. Склонуйте репозиторій за допомогою команди git clone
   https://github.com/Daniel-Slavnyi/test-products.git
2. Відкрийте папку проекту та виконайте команду npm install, щоб встановити всі
   залежності
3. Запустіть додаток командою npm start

Додаток буде доступний за адресою http://localhost:3000/.
